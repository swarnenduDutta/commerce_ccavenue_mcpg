DESCRIPTION===========Provides CCAvenue MCPG payment gateway integration with Commercecommerce_ccavenue_mcpg.module : the core API module
INSTALLATION============1 . Go to the admin/site settings/modules2 . click the Install New Module button. choose the CCAvenue MCPG module and install it.3 . Enable the CCAvenue MCPG module.4 . Goto admin/store settings/payment methods and create a rule for CCAvenue5 . Edit Action and Configure the Merchant id,  Access Code, Encryption key.
